# 랭크(Rank) - 차원의 수, 셰이프(Shape) - 요소의 갯수

```python
3 # 랭크가 0인 텐서(Tensor), 셰이프는 []
[1. ,2., 3.] # 랭크가 1인 텐서, 셰이프는 [3]
[[1. ,2., 3.], [4., 5., 6.]] # 랭크가 1인 텐서, 셰이프는 [2, 3]
[[[1. ,2., 3.]], [[7., 8., 9.]]] # 랭크가 1인 텐서, 셰이프는 [2, 1, 3]
```
랭크가 0이면 스칼라, 1은 벡터, 2는 행렬, 3 이상이면 n차원 텐서
